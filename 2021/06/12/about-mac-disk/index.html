关于Mac && NTFS格式移动硬盘
Created 2021-06-12| Updated 2021-06-12
Post View:

本人的电脑为Mac OS，当初买移动硬盘的时候为了在Mac和Windows之间通用就采用了NTFS格式，于是衍生出了接下来的问题。

第一个问题：读写问题

  • 问题陈述：NTFS格式磁盘因为某些原因，在Mac上是只读格式，无法写入文件。但其实并不是无法写入，只是没有开写权限。
  • 解决方法：写文件系统配置文件 /etc/fstab，具体操作如下：
    • 命令行输入 sudo vim /etc/fstab
    • 添加一行配置 LABEL=yourDiskName none ntfs rw,auto,nobrowse
    • 将磁盘拔出重新插到电脑，此时桌面上不会出现它的图标，但其实依然能通过访达访问到
    • 使用访达的 go to folder 功能，输入路径 /Volumes，回车，就能看到和 Macintosh HD磁盘同一个目录下的移动硬盘
    • 为了访问方便可以添加软链接

参考链接：

  • 使用mount_ntfs指令

第二个问题：无法挂载

  • 问题陈述：上一次使用还好好的，隔了几天之后就无法挂载了，去掉之前的配置之后可以挂载，但又回到了只读的格式。
  • 解决方法：上网查询后发现可能是自己前段时间将磁盘借给过其他同学，她/他（也可能是我自己）没有正常弹出我可爱的磁盘。可能就是因为这个磁盘出现了一些错误，但在mac上无法修复，只能使用Windows自带的 chkdsk 指令，具体操作如下：
    • 将磁盘插入Windows电脑，查看自己的硬盘名称（以G盘为例）
    • 打开终端，输入 chkdsk G: /f ，这里的 /f 参数功能即为修复磁盘上的错误
    • 如果显示有程序占用之类的问题，直接选择强制卸除即可（那个程序很可能是360…）
    • 修复完毕后再插入mac，应该就能正常读写啦～
